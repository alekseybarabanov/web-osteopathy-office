Политика конфиденциальности

Данное приложение использует Google Calendar API исключительно в режиме чтения для отображения событий календаря.

Приложение не собирает, не хранит и не передает персональные данные третьим лицам.

Данные Google-аккаунта используются только для авторизации доступа к календарю и хранятся локально в браузере пользователя.
